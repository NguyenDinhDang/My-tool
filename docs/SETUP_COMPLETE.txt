  TOOL 1: MARKDOWN TO WORD (md2word)


What it does:
  • Converts .md files to professional .docx documents
  • Preserves formatting: headings, code, tables, links
  • Adds proper styling and colors

Usage:
  python agi_cli.py md2word <file.md> [-o output.docx]

Example:
  python agi_cli.py md2word myfile.md -o myfile.docx

Test it:
  python agi_cli.py md2word sample.md

 TOOL 2: AUTO-FILL GOOGLE FORM (autofill)


What it does:
  • Automatically fills Google Forms with random responses
  • Simulates human-like typing
  • Bypasses bot detection
  • Handles checkboxes, radio buttons, and text fields

Usage:
  python agi_cli.py autofill [-n number_of_submissions]

Example:
  python agi_cli.py autofill -n 10
