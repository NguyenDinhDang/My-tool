# AGI - Automation & Conversion CLI Tool

Tai lieu chinh nam o `../README.md`.

Du an cung cap CLI cho hai nhom tac vu:

- Chuyen Markdown sang Word `.docx`
- Tu dong dien Google Form khi co quyen thuc hien

Chay nhanh tu thu muc goc:

```bash
pip install -r requirements.txt
python agi.py --help
python agi.py md2word samples/mermaid_sample.md
```
